# LeetCode Solutions

Repositório criado para organizar minhas soluções de exercícios do LeetCode.

O objetivo é praticar lógica de programação, estruturas de dados, algoritmos e SQL, mantendo um histórico público de evolução como desenvolvedor.

## Objetivos

- Praticar resolução de problemas.
- Melhorar raciocínio lógico.
- Estudar estruturas de dados e algoritmos.
- Documentar soluções com explicações simples.
- Construir consistência no GitHub.
